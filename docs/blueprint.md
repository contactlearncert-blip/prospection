# **App Name**: ProspectZen

## Core Features:

- Automated Prospect Discovery: Automatically search the internet for potential leads based on industry, location, and online presence, using an LLM tool to evaluate each potential lead.
- Contact Information Collection: Collect contact information (email, phone number, contact forms) from identified prospects in a secure and compliant manner.
- Prospect Dashboard: Display prospects in a clear dashboard with filtering and sorting capabilities based on criteria like industry, location, and contact date.
- Personalized Messaging: Send customized messages to prospects directly from the application.
- Prospect Status Tracking: Track the status of each prospect: contacted, replied, interested, not interested.
- Analytics and Reporting: Generate statistics on prospects and actions performed to evaluate lead generation efforts.

## Style Guidelines:

- Primary color: A calming blue (#5DADE2) to inspire trust and reliability.
- Background color: A light gray (#F0F4F7), a desaturated tint of the primary blue, providing a neutral backdrop that enhances the focus on the content.
- Accent color: A vibrant green (#82E0AA) for positive actions, such as 'contacted' or 'interested', adding a fresh and engaging feel.
- Body and headline font: 'PT Sans' for a modern, readable, and friendly user experience.
- Use simple, professional icons to represent different industries and prospect statuses.
- Clean and intuitive dashboard layout for easy navigation and prospect management.
- Subtle transitions and animations to provide feedback and improve user engagement.
